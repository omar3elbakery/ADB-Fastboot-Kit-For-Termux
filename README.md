# ADB-Fastboot-Kit-For-Termux

# Installation 
- 1-install termux from f-droid
```env
https://f-droid.org/repo/com.termux_1002.apk
```
- 2-install termux-api from f-droid
```env
https://f-droid.org/repo/com.termux.api_1002.apk
```
- 3-open termux-api and give it permissions
- 4-open termux and run this command to grant storage permission ->
```env
termux-setup-storage
```
5-
```env
pkg install wget -y
```
6-
```env
 wget https://github.com/omar3elbakery/ADB-Fastboot-Kit-For-Termux/releases/download/ADB%26Fasboot_Kit/install.sh && bash install.sh
```

# USAGE 
```env
kit
```
or 
```env
bash kit
```
